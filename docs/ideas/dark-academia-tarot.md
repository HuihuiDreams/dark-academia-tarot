# 🔮 暗黑学院风桌面塔罗微件 (Dark Academia Tarot Widget)
**Idea Refinement One-Pager**

## 1. 问题声明 (Problem Statement)
> **How Might We:** 我们如何创造一个轻量、纯净、无边框悬浮于 macOS 桌面的“暗黑学院风”神秘学交互微件，让暗黑美学与玄学爱好者在日常思考、情绪内耗或寻求灵感时，通过极具物理沉稳感与仪式感的“翻转牌阵-提问”交互，并借助 Gemini API 的心理投射解读，获得深度的情绪疗愈与美学陪伴？

## 2. 推荐方案方向 (Recommended Direction)
将该桌面微件定位为**「常驻桌面的暗黑学院派心理与玄学日记摆件」**。

- **极简无边框悬浮形态**：基于 Tauri + Vanilla TypeScript + Vanilla CSS 构建。去除所有系统窗口边框，开启 macOS 窗口透明度 (`transparent: true`)，支持全窗口鼠标任意拖拽 (`data-tauri-drag-region`) 与右键快捷菜单（重新洗牌、设置 API Key、退出等）。
- **古老图书馆美学 (Dark Academia Aesthetics)**：采用深邃古铜金 (`#c5a059`)、羊皮纸暗黄 (`#e6d5ac`)与黑藏青 (`#121318`) 为核心调色盘。配合古典衬线字体（如 Playfair Display / Cormorant Garamond / Noto Serif SC）。
- **阻尼感物理交互**：3D CSS `perspective: 1200px` 配合缓动贝塞尔曲线 (`cubic-bezier(0.34, 1.56, 0.64, 1)`)，使每一次卡牌翻转都带有沉稳厚重的阻尼感与物理弹性回弹，并触发金色微光/墨水涟漪特效。
- **22 张大阿卡纳牌库与真卡背**：预设完整的 22 张大阿卡纳卡牌数据（包含正逆位、暗黑学院风释义、心理学对应原型）。其中核心牌面与卡背由本地美术脚本生成/构建，支持随机抽取 3 张（对应「过去/执念」-「当下/假象」-「未来/钥匙」）。
- **流式心理投射解读 (Gemini Streaming)**：使用 `@google/generative-ai` SDK 调用 `gemini-1.5-flash` 模型。在三张牌全翻且用户提交心事/问题后，带有占星烛光微晃的 Loading 状态闪烁，随后以复古打字机流式渐显出具有文学质感与情绪价值的解读。

## 3. 核心假设验证 (Key Assumptions to Validate)
- [ ] **窗口透明与拖拽体验**：macOS 下 Tauri 的无边框透明窗口在各种壁纸下视觉对比度足够，拖拽流畅且不会吞掉卡牌点击事件。
- [ ] **交互回弹阻尼质感**：纯 CSS 3D Transforms 配合 JS 事件能完美还原真实纸牌的厚重感与翻牌快感。
- [ ] **Gemini 流式打字机沉浸感**：`gemini-1.5-flash` 的流式响应速度 (`generateContentStream`) 能在 1~2 秒内给出首字，配合羊皮纸展开动效不让用户感到卡顿。

## 4. MVP 范畴 (MVP Scope - 绝对聚焦)
- **基建 (Step 1)**：Tauri + Vite + Vanilla TS 项目脚手架，配置无边框与全局拖拽。
- **美术 (Step 2)**：编写卡牌素材生成/准备脚本，准备或生成极具暗黑学院风的经典卡背及主牌面插图，构建 22 张大阿卡纳 JSON 数据字典。
- **界面 (Step 3)**：桌面中央悬浮卡牌陈列架（3 张覆面卡牌）+ 墨水半透明输入框。点击卡牌平滑翻转，展现随机牌面与正逆位。
- **AI 解读 (Step 4)**：三牌全翻后展示发光连线，结合用户输入的疑惑发出请求。前端实时接收流式文本并打字机渲染。

## 5. 暂不处理的复杂功能 (Not Doing - and Why)
- ❌ **复杂的在线用户注册与云端数据同步**：作为纯净个人桌面电子摆件，所有配置（API Key、历史解读记录）均存储在本地 localStorage/JSON 中，保证绝佳隐私与极致启动速度。
- ❌ **每次发问实时根据输入在线生成 3 张新卡牌图**：实时生成 3 张高画质图片通常需要 15~30 秒，会破坏抽牌解惑的即时流畅心流；改用本地 22 张高质量牌库随机抽取 + 文本流式解读，2 秒内即可得到沉浸反馈。
- ❌ **庞大的 3D 引擎 (Three.js/WebGL)**：为确保桌面微件常驻内存占用在几十 MB 以内（而非几百 MB），坚持采用硬件加速的 3D CSS 变形来实现最高性能的动效。

## 6. 待解决问题 (Open Questions)
- 用户的 Gemini API Key 默认通过界面上的「钥匙孔/齿轮小图标」打开复古设置面板进行安全配置与本地保存。
